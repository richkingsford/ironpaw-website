# IronPaw Website

Static homepage for IronPaw, an autonomous construction robotics company.

## Structure

- `index.html` - single-page homepage
- `styles.css` - responsive visual system
- `assets/` - optimized web images and source reference images

## Local Preview

Open `index.html` directly in a browser, or run a simple static server:

```powershell
python -m http.server 4173
```

## Cloudflare Pages

Use Cloudflare Pages with Direct Upload or connect this repo. No build command is required and the output directory is the repository root.
